The Fine Print
AI-Powered Policy Analysis

"The Fine Print" is a full-stack web application designed to bridge the gap between complex government legislation and civic understanding. By leveraging the power of Gemini 1.5 Flash, the platform translates dense legal jargon into plain English, predicts future societal impacts, and provides actionable steps for the everyday citizen.

Technical Stack
Frontend: React.js, Tailwind CSS

Backend: FastAPI (Python), Uvicorn

AI Engine: Google Gemini 1.5 flash

Environment: Python-Dotenv for secure API management

Frontend:

Bash
